# Calendar

A pixel- and interaction-accurate rendition of the iOS Calendar app, built with Next.js
and driven by an ICS-style (iCalendar) data model.

See [`docs/ios-calendar-ui-spec.md`](docs/ios-calendar-ui-spec.md) for the detailed UI
and interaction spec this build targets.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
